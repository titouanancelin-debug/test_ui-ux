## Comment exporter depuis Canva

1. Ouvrir le design
2. Clic sur **Partager** → **Télécharger**
3. Choisir le format : PDF (impression) ou PNG/JPEG (web)
4. Déposer le fichier exporté dans le bon dossier ci-dessus
