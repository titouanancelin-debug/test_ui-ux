# Photos

Classer les photos dans les sous-dossiers :

- `spectacles/` → photos de plateau, représentations
- `ateliers/` → photos des ateliers en cours
- `equipe/` → portraits de l'équipe artistique
- `lieu-filature/` → photos de la Filature de l'Isle (salle, espaces)

Nommage recommandé : `[categorie]_[description]_[année].jpg`
Exemple : `spectacles_gens-de-peu_repetition_2025.jpg`
