# Identité Graphique — Cie Rouletabille Théâtre
## Saison 2025–2026

Déposer ici tous les fichiers visuels de la compagnie.

---

### Structure des dossiers

| Dossier | Contenu |
|---|---|
| `01-logo/` | Fichiers logo en couleur, blanc, noir (PNG, SVG, PDF) |
| `02-couleurs-typographie/` | Palette de couleurs, guide typo, charte graphique |
| `03-photos/` | Photos classées par catégorie |
| `04-supports-imprimes/` | Affiches, flyers, programmes de salle |
| `05-web-reseaux/` | Visuels pour le site, Instagram, Facebook |
| `06-canva/` | Liens et exports des designs Canva |

---

### Formats recommandés
- **Logo** : SVG (vectoriel) + PNG 1000px minimum
- **Photos** : JPEG, 72 dpi pour le web / 300 dpi pour l'impression
- **Affiches** : PDF haute résolution + JPEG aperçu
