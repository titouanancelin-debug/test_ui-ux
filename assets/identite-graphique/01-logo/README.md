# Logo — Cie Rouletabille Théâtre

## Versions disponibles

| Fichier | Usage |
|---|---|
| `couleur/logo-couleur.svg` | Version principale — fond blanc/clair |
| `couleur/mark-seul.svg` | Pictogramme seul (physalis) sans texte |
| `blanc/logo-blanc.svg` | Sur fond sombre (aubergine, noir) |
| `noir/logo-noir.svg` | Impression noir & blanc |

## Règles d'usage

- **Espace de protection** : laisser un espace égal à la hauteur du mark autour du logo
- **Taille minimale** : 120px de large (web) / 30mm (impression)
- **Ne pas** déformer, recolorer, ou ajouter d'ombre portée
- **Fond recommandé** : `#F4E8D5` (papier), `#FFFFFF`, ou `#3A1B2E` (aubergine)

## Formats à exporter depuis SVG

- **PNG 1000px** pour le web et les réseaux sociaux
- **PNG 500px** pour les miniatures et favicons
- **PDF vectoriel** pour l'impression
